# FileTrace - 个人文件与笔记管理系统

一个现代化的文件管理和笔记系统，让您轻松保存、查看和管理各种类型的文件，同时支持为文件添加笔记。

## ✨ 主要特性

### 🗃️ 文件管理
- **多格式支持**: 支持图片、视频、PDF、音频、文档等多种文件格式
- **智能预览**: 在线预览图片、视频、PDF等文件，无需下载
- **文件分类**: 按文件类型自动分类，支持筛选和搜索
- **拖拽上传**: 便捷的拖拽上传体验
- **批量操作**: 支持批量上传和管理
- **文件夹上传**: 支持上传整个文件夹，自动保留目录结构
- **文件夹结构**: 智能创建和维护文件夹层级关系

### 📝 笔记系统
- **富文本编辑**: 支持Markdown格式的笔记编辑
- **文件关联**: 为文件添加笔记，支持在笔记中引用文件
- **标签管理**: 通过标签整理和分类笔记
- **快速搜索**: 全文搜索笔记内容

### 🎨 界面设计
- **简约美观**: 现代化的用户界面设计
- **响应式布局**: 完美适配桌面和移动设备
- **深色模式**: 支持明暗主题切换
- **流畅动画**: 精心设计的交互动画

### 🔍 高级功能
- **智能搜索**: 支持文件名和笔记内容的全文搜索
- **数据统计**: 直观的数据统计面板
- **快捷操作**: 便捷的快捷键和右键菜单
- **历史记录**: 文件上传和笔记修改历史

## 🚀 技术栈

### 前端
- **Vue.3**: 现代化的前端框架
- **Vue Router**: 单页面应用路由
- **Tailwind CSS**: 实用优先的CSS框架
- **DaisyUI**: 基于Tailwind的组件库

### 后端
- **FastAPI**: 现代化的Python Web框架
- **SQLAlchemy**: 强大的ORM工具
- **Pydantic**: 数据验证和序列化
- **JWT**: 安全的用户认证

### 存储
- **SQLite/PostgreSQL**: 数据库存储
- **本地文件系统**: 文件存储
- **AWS S3** (可选): 云存储支持
